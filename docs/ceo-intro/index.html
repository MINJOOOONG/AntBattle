04 Customer

타겟 고객

주식에 관심은 있지만 실제 투자는 부담스러운 2030 개인 투자자입니다.

친구들과 "오늘 뭐 오를 것 같냐", "내가 말한 종목 올랐지?" 같은 대화를 나누지만, 그 예측을 게임처럼 기록하고 겨룰 공간은 부족합니다.

2030 주식 관심층부담 없이 수익률 감각을 확인하고 싶은 사용자
친구 그룹 / 단체방누가 더 잘 맞혔는지 기록하고 놀고 싶은 사용자
캐릭터·랭킹 선호층결과를 성장, 꾸미기, 소셜 자랑으로 연결하고 싶은 사용자
05 Market Gap

기존 대안의 한계

증권 앱은 거래 도구이고, 모의투자는 연습 중심입니다.

개미배틀은 "주식 정보 제공"이 아니라 "주식 대화의 게임화"라는 빈틈을 공략합니다.

1증권 앱: 초보자에게 어렵고 친구와 노는 경험이 약함
2모의투자: 장기 운영 중심이라 짧은 재미가 부족함
3단체방 대화: 기록, 판정, 보상, 랭킹 없이 말로만 끝남
4주식 커뮤니티: 친구 관계 기반 1:1 게임 루프가 약함
06 Product

제품 개요

배틀 신청부터 개미콩 보상, 캐릭터 꾸미기로 이어지는 반복 루프입니다.

사용자는 실제 돈을 투자하지 않고도 시장 데이터를 기반으로 한 긴장감과 재미를 경험할 수 있습니다.

07 Feature Set

핵심 기능

모바일 앱과 API 서버를 중심으로 실제 사용 가능한 배틀 경험을 구성합니다.

회원가입, 친구 시스템, 1:1 배틀, 한국 주식 데이터, 상점, 랭킹, 마이페이지까지 하나의 게임 루프로 연결합니다.

Mobile App홈·배틀·상점·친구·마이페이지 화면
API ServerExpress, Prisma, PostgreSQL 도메인 API
Battle Loop신청, 수락, 종목 선택, 결과 판정
Reward개미콩, 상점, 랭킹, 캐릭터 꾸미기
08 Differentiation

차별성

거래 도구가 아니라, 친구들과 주식으로 노는 경험을 만듭니다.

모의 수익률 배틀, 개미콩, 개미상점, 캐릭터 꾸미기, 랭킹 등급으로 숫자 중심의 투자 결과를 게임적 보상으로 번역합니다.

09 Business Model

비즈니스 모델

더 재미있게 놀고 내 개미를 표현하기 위한 결제 구조입니다.

개미콩은 현금성 보상으로 바뀌지 않으며, 수익화는 꾸미기, 시즌 콘텐츠, Pro 구독, 캐릭터 IP 확장에서 발생합니다.

개미콩 충전참여와 보상의 중심 재화
꾸미기 아이템캐릭터 애착 기반 구매
시즌 패스랭킹과 미션 보상 확장
Pro 구독추가 배틀, 프리미엄 통계
10 Growth Strategy

고객 획득과 로드맵

초기에는 2030 투자 관심층에서 베타를 검증합니다.

친구에게 배틀 신청, 결과 카드 공유, 내 개미 자랑하기를 통해 초대 전환과 반복 배틀을 만듭니다.

1지인 그룹과 투자 관심층 중심의 초기 베타 운영
2시즌 랭킹, 결과 카드 공유, 프라이빗 리그 도입
3꾸미기 아이템, 시즌패스, Pro 구독 수익화 실험
4캐릭터 굿즈, 브랜드 협업, 팝업 스토어 확장

12 Vision

비전

"주식으로 노는 귀여운 개미들의 배틀장"을 만듭니다.

친구 초대와 반복 배틀이 자연스럽게 일어나는 소셜 투자 놀이 공간. 장기적으로 개미 캐릭터 IP와 시즌 시스템, 프라이빗 리그, 결과 공유, 굿즈와 팝업 스토어까지 확장합니다.

개미배틀은 투자 추천, 투자 자문, 자동매매, 금전 베팅을 제공하지 않습니다. 표시되는 수익률과 종목 정보는 게임/학습 목적의 콘텐츠입니다.
